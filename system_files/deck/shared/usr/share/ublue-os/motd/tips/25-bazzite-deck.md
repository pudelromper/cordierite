*Want to install Decky Loader?* There's a `ujust` command for that! `ujust setup-decky install`
The updater built into 󰓓 Steam has been modified to update Cordierite, Flatpaks, and Distrobox containers. *Just ignore the changelog.*
*Install an app with Lutris?* Right click on it and `Create steam shortcut` to easily access it from 󰓓 Steam.
*Apps missing icons?* The `SteamGridDB` plugin for Decky Loader makes it easy to add missing art~[More info](https://github.com/SteamGridDB/decky-steamgriddb)
*Looking for more security?* Netbird VPN is built in, and can be managed with `ujust` commands.
*Using a handheld that doesn't have enough buttons?* 󰓓 Steam now supports touch gestures to slide out the menus, thanks to [ChimeraOS](https://chimeraos.org/).
